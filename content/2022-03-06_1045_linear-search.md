# Linear Search

Linear search, which grows linearly (as e grows, runtime grows at the same rate):

```python
def search(L, e):
    for elem in L:
        if elem == e:
            return True
        if elem > e:
            return False
    return False
```

[[2022-03-06_1050_binary-search]] is an improvement provided list is sorted.
